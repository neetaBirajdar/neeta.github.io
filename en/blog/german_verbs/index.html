Verbs are really important if you want to learn and understand any language. In this blog, I have added some basic examples of how Regular and Irregular verbs look like with different forms of Nouns.

What is a Verb:

A verb is a word that in syntax generally conveys an action, an occurrence, or a state of being. (As per: https://en.wikipedia.org/wiki/Verb )

Verbs with Examples:

A) Regular Verbs:

Note:

The verb takes different form when comes with different pronouns:

  • Ich ( I ) : verb + e
  • Du ( you [informal ]) : verb + st
  • Es/Er/Sie ( it/he/she ) : verb + t
  • Wir ( we [pl ] ): verb + en
  • Ihr ( you [pl ] ) : verb + t
  • Sie/sie ( you [formal ] / they [pl ] ) : verb + en
  1. Gehen : To go
  2. Trinken : To drink
  3. Machen : To make
  4. Schalen : To sleep
  5. Brauchen : To need

Conjugation :

Subject To Go To Drink To Make To Sleep To Need
Ich gehe trinke mache schlafe brauche
Du gehst trinkst machst schlafst brauchst
Er/Sie/Es geht trinkt macht schlaft braucht
Wir gehen trinken machen schlafen brauchen
Ihr geht trinkt macht schlaft braucht
Sie/sie gehen trinken machen schlafen brauchen

Example:

  • Ich gehe ins Kino ( I go to the cinema )
  • Was machst du da? ( What are you doing? )
  • Er trinkt Wasser ( He drinks water )
  • Wir schlafen in der Nacht ( We sleep in the night )
  • Du gehst zur Schule ( You go to school )
  • Sie/sie brauchen Orangensaft ( They need Orange Juice )

B) Irregular Verbs:

Note:

The verb takes different form when comes with different pronouns:

  • Ich ( I ) : verb + e
  • Du ( you [informal ]) : verb + e -> i + st
  • Es/Er/Sie ( it/he/she ) : verb + e -> i + t
  • Wir ( we [pl ] ): verb + en
  • Ihr ( you [pl ] ) : verb + + t
  • Sie/sie ( you [formal ] / they [pl ] ) : verb + en
  1. Essen : To eat
  2. Lesen : To read
  3. Sprechen : To speak
  4. Nehmen : To take
  5. Treffen : To meet
